# 📘 Assignment: Python Text Processing

## 🎯 Objective

Master string manipulation, file I/O operations, and text processing techniques in Python. You'll build practical tools for reading, analyzing, and transforming text files. This assignment develops essential skills for data processing, text analysis, and file handling that are fundamental to real-world programming tasks.

## 📝 Tasks

### 🛠️ Work with Strings and Text Manipulation

#### Description
Practice fundamental string operations and manipulation techniques to process text data effectively.

#### Requirements
Completed program should:

- Perform string operations (concatenation, slicing, case conversion)
- Use string methods (split, strip, replace, find, count)
- Search for patterns or substrings within text
- Format strings using f-strings or .format()
- Validate and clean text input (remove whitespace, normalize spacing)
- Demonstrate string indexing and iteration


### 🛠️ Read Files and Process Text Data

#### Description
Implement file I/O operations to read text files and extract meaningful information from them.

#### Requirements
Completed program should:

- Open and read files using appropriate file modes
- Handle different file formats (plain text, CSV-like data)
- Process files line-by-line efficiently
- Extract specific data from file contents
- Store processed data in appropriate data structures
- Handle file not found errors gracefully


### 🛠️ Analyze Text and Create Output Files

#### Description
Perform text analysis operations and write results to new files, combining reading and writing capabilities.

#### Requirements
Completed program should:

- Analyze text properties (word count, character count, unique words)
- Process and transform text based on specific criteria
- Generate summary statistics or reports from text files
- Write processed results to a new output file
- Format output clearly with headers, labels, and organized data
- Implement sorting or filtering of text analysis results
