# 📘 Assignment: Python Data Structures

## 🎯 Objective

Master the fundamental data structures in Python: lists, dictionaries, sets, and tuples. You'll learn when to use each structure, how to manipulate collections of data efficiently, and how to choose the right data structure for different programming tasks. These core data structures are essential building blocks for all Python programming.

## 📝 Tasks

### 🛠️ Work with Lists and Tuples

#### Description
Learn to create, access, modify, and iterate through lists and understand when to use immutable tuples.

#### Requirements
Completed program should:

- Create lists with multiple data types and access elements by index
- Use list methods: append, insert, remove, pop, sort, reverse, extend
- Perform list slicing and use negative indexing
- Create tuples and access tuple elements
- Understand the difference between mutable lists and immutable tuples
- Use list comprehensions to create and filter lists
- Demonstrate unpacking of list and tuple elements


### 🛠️ Use Dictionaries for Key-Value Data Storage

#### Description
Work with dictionaries to store and retrieve data using keys, and perform common dictionary operations.

#### Requirements
Completed program should:

- Create dictionaries and add/modify key-value pairs
- Access values using keys and handle missing keys safely
- Use dictionary methods: keys(), values(), items(), get(), pop(), update()
- Iterate through dictionaries by keys, values, or both
- Use dictionaries to count occurrences (frequency counting)
- Nest dictionaries for complex data structures
- Convert between lists and dictionaries


### 🛠️ Apply Sets and Choose Appropriate Data Structures

#### Description
Use sets for unique values and membership testing, and demonstrate choosing the right data structure for different scenarios.

#### Requirements
Completed program should:

- Create sets and add/remove elements
- Use set operations: union, intersection, difference, symmetric difference
- Verify membership in sets efficiently
- Remove duplicates from lists using sets
- Understand when to use sets vs lists vs dictionaries
- Combine different data structures to solve problems
- Demonstrate performance considerations of each data structure
